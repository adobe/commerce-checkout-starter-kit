# Checkout Payment Method

Out-of-process payment method validation and filtering for the Adobe Commerce checkout starter
kit, built as an independent Adobe Commerce App Management app.

See [Payment in Checkout Starter Kit](https://developer.adobe.com/commerce/extensibility/starter-kit/checkout/payment-install)
for the business context, and the [App Management docs](https://developer.adobe.com/commerce/extensibility/app-management/)
for install, build/deploy, and association.
