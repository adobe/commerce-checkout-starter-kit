# Checkout Totals Collector

Starter app scaffolding for the out-of-process discount webhook actions (cart totals collector
rules), built as an independent Adobe Commerce App Management app.

See [Totals Collector in Checkout Starter Kit](https://developer.adobe.com/commerce/extensibility/starter-kit/checkout/totals-collector-install)
for the business context, and the [App Management docs](https://developer.adobe.com/commerce/extensibility/app-management/)
for install, build/deploy, and association.
